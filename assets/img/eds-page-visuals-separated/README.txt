EDS separated page visuals

This package contains the separated visual panels from the approved composite image.
All images are text-free and keep the same dark industrial style with green EDS accents.

Folders:
- png/: exact cropped PNG images
- webp/: web-optimized WebP versions
- manifest.json / manifest.csv: suggested page usage and image sizes

Note:
The original composite contained 20 panels. These files are therefore the separated version of those 20 visuals.
